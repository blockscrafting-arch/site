## Проблема

Хотелось своего Джарвиса: «напомни», «запиши идею», «добавь встречу», «найди» — без разорительных подписок и без полдня на настройку.

## Решение

Собрал контур за один вечер: бот в Telegram (BotFather) + n8n как backend + любая LLM (OpenAI / Gemini / Claude) + инструменты (календарь, таскер, имейл, поиск) + векторная база для памяти. Голосовые сообщения транскрибируются и обрабатываются; бот выполняет команды и запоминает контекст.

## Результат

**Second Brain** в кармане: скинул войс на бегу — «Запиши идею по посту, напомни через 3 часа и закинь встречу на завтра» — бот записал, поставил напоминание и создал событие. Всё без ручного ввода и без ежемесячных абонплат за корпоративные решения.

## Стек

Telegram, n8n, OpenAI / Gemini / Claude (на выбор), календарь/таскер/имейл, векторная память.
