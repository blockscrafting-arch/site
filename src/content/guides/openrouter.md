## Шаги

1. Зайдите на [https://openrouter.ai/](https://openrouter.ai/).
2. Наведите на аватарку → **keys** → **Create API Key** → любое имя → **create** → кнопка **копировать**. Ключ формата: `sk-or-v1-5b36d...`.
3. Пополнение: в том же окне слева в меню **Credits** → **add credits**.

Иностранную карту можно завести, например, в Telegram-боте: @zarub_robot.
